# Refresh PWA Assets

Regenerate all PWA icon PNGs, favicon, and manifest screenshots for the Lingua app.

## What this skill produces

**Icons** (from `static/icons/lingua.svg`):

- `static/icons/icon-192.png` (192x192)
- `static/icons/icon-512.png` (512x512)
- `static/icons/icon-512-maskable.png` (512x512, padded on brand background)
- `static/favicon.png` (48x48)
- `static/favicon.ico` (32x32 ICO)
- `static/apple-touch-icon.png` (180x180)
- `static/apple-touch-icon-precomposed.png` (180x180)

**Screenshots** (from the running preview server):

- `static/screenshots/desktop.png` (1280x720)
- `static/screenshots/mobile.png` (780x1688 — 390x844 viewport at 2x DPR)

## Steps

### 1. Generate icons

Run the existing icon generation script:

```bash
npx tsx scripts/generate-icons.ts
```

This reads `static/icons/lingua.svg` and outputs all PNG sizes plus the favicon.

### 2. Build and start the preview server

The screenshots need a running app. Build and start preview:

```bash
npm run build && npm run preview
```

Run this in the background so the server stays up while you take screenshots. Wait a few seconds for it to be ready.

### 3. Take desktop screenshot

Use the Puppeteer MCP tools in this order:

1. **Launch browser** with desktop viewport:
   - `puppeteer_launch` with viewport `{ width: 1280, height: 720, deviceScaleFactor: 1 }`
   - headless: true

2. **Create a page**: `puppeteer_new_page` with pageId `"desktop"`

3. **Navigate**: `puppeteer_navigate` to `http://localhost:4173` with waitUntil `"networkidle0"`

4. **Screenshot**: `puppeteer_screenshot` with pageId `"desktop"` and path set to the **absolute path** to `static/screenshots/desktop.png`

### 4. Take mobile screenshot

1. **Close the browser**: `puppeteer_close_browser`

2. **Launch browser** again with mobile viewport:
   - `puppeteer_launch` with viewport `{ width: 390, height: 844, deviceScaleFactor: 2, isMobile: true, hasTouch: true }`
   - headless: true

3. **Create a page**: `puppeteer_new_page` with pageId `"mobile"`

4. **Navigate**: `puppeteer_navigate` to `http://localhost:4173` with waitUntil `"networkidle0"`

5. **Screenshot**: `puppeteer_screenshot` with pageId `"mobile"` and path set to the **absolute path** to `static/screenshots/mobile.png`

6. **Close browser**: `puppeteer_close_browser`

### 5. Stop the preview server

Kill the background preview server process.

### 6. Verify

Confirm all files were generated by listing:

- `static/icons/` — should have `icon-192.png`, `icon-512.png`, `icon-512-maskable.png`, `lingua.svg`
- `static/screenshots/` — should have `desktop.png`, `mobile.png`
- `static/favicon.png`, `static/favicon.ico`, `static/apple-touch-icon.png`

Report the file sizes to the user so they can sanity-check the outputs.
